About this demo

  • This presentation demonstrates the three quiz functions from the rquiz package — singleQuestion(), multiQuestions(), and fillBlanks() — with quizzes about marine biology.
  • Each quiz type is shown twice: first with default settings, then with a custom dark ocean theme applied via rquizTheme().

Tips:

  • Slidy slides automatically expand to fit the quiz content, so scroll = TRUE is usually not needed.
  • To style text within questions (e.g. species names in italics), use HTML tags such as <em> (italics), <strong> (bold), <code> (inline code), or <span> with inline CSS directly in the question or answer option strings.
